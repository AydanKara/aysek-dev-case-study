A public case study for **aysek.dev**, my personal portfolio and freelance web development website.

The original project repository is private because it contains business logic, admin functionality and project-specific implementation details. This public repository presents the concept, goals, features and technical direction of the project without exposing private source code.

---

## Overview

**aysek.dev** is a modern portfolio and business website built to present my work as a Front-End Developer. The website is designed for small businesses, local clients and potential freelance partners who need professional, responsive and easy-to-manage websites.

The goal of the project was to create more than a simple portfolio. It functions as a business presence, project showcase, contact channel and technical demonstration of modern web development practices.

**Live website:** [https://aysek.dev](https://aysek.dev)

- Present my web development services in a professional and trustworthy way

## Design Direction

The design direction focuses on a modern, elegant and professional look that fits a freelance web development brand.

Key design principles:

- Clean layouts with strong spacing
- Modern typography
- Clear contrast
- Subtle animations
- Polished project previews

## Architecture Highlights

The private production project uses a modular React architecture with separate areas for:

- Reusable UI components
- Page-level route components
- Admin components
- Contact form logic
- Search functionality
- Error handling
- Language/context management
- Supabase integration
- Utility and performance helpers

This structure helps keep the project maintainable as more services, case studies and business features are added.

This repository is intentionally a **case-study repository**.
